# Olympus.js

> A modular BIM framework for web-based AECO sofwtare

Olympus.js is a JavaScript library that brings Building Information Modeling (BIM) capabilities to the browser. Built for architects, engineers, and construction professionals who need powerful web-based tools for working with building data.

## Features

- 🏗️ **BIM-First Design** - Purpose-built for AEC industry workflows
- 📊 **Centralized Data Store** - Manage building information and project data
- 🎨 **3D Visualization** - Integrated Three.js editor for model viewing
- 🔧 **Extensible Operators** - Command pattern for BIM operations
- 🎯 **Smart Tooling** - Context-aware tools that adapt to your project
- 📝 **Visual Programming** - Node editor for creating BIM workflows
- 🐍 **Python Support** - Embedded Python via Pyodide for advanced scripting
- 📐 **IFC Native** - Full support for Industry Foundation Classes
